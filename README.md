# GanBan
## Build with vue CLI
---
### **Project Setup**
* **Change directory to client**
  ```
  $ cd client/
  ```
* **Install Depencies**
  ```
  $ npm install
  ```
---
### **Project Operation**
* **Run On Your Local for Development**
  ```
  $ npm run serve
  ```
* **Production**
  ```
  $ npm run build
  ```
